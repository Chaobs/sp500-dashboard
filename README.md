## ✨ 项目亮点

### 🎯 核心优势

- ✅ **无需代理**: 使用国内可访问的数据源,无需任何代理即可使用
- ✅ **开箱即用**: 纯前端实现,双击HTML文件即可打开,无需安装依赖
- ✅ **实时数据**: 接入东方财富API,数据实时更新,准确可靠
- ✅ **适合定投**: 自动判断牛熊市,提供200日均线参考,辅助定投决策
- ✅ **轻量快速**: 不依赖外部库,使用原生Canvas绘制,加载速度极快

## 📊 功能特性

### 1. 实时数据监控
- **S&P 500 指数**: 当前价格、涨跌幅、涨跌额
- **VIX 恐慌指数**: 市场波动率指标,情绪参考
- **牛熊判断**: 基于200日均线自动判断市场趋势

### 2. 价格走势图
- 7天价格走势线 (蓝色)
- 200日均线参考 (黄色虚线)
- 当前价格标记 (绿点)
- 最高/最低价格标注

### 3. 新闻聚合
- 8条最新财经新闻
- 来源: 新浪财经、东方财富、同花顺、雪球等
- 自动更新

## 📈 使用场景

### 💰 适合定投人群
- **自动判断牛熊**: 价格 > 200日均线 = 牛市 🐂, 价格 < 200日均线 = 熊市 🐻
- **情绪参考**: VIX指数辅助判断市场恐慌程度
- **趋势跟踪**: 7天走势图快速了解市场动向

### 📊 适合日常看盘
- **实时数据**: 每30秒自动更新价格
- **新闻聚合**: 一站式了解市场动态
- **无需翻墙**: 国内网络直连访问

### 数据来源
- **API**: 东方财富

### 数据准确性
| 指标 | 准确度 | 说明 |

| VIX | ⭐⭐⭐☆☆ | 估算值 |
